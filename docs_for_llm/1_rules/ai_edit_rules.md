# AI Edit Rules

1. AI MUST NOT delete any content.
2. AI MUST NOT overwrite existing content.
3. AI MAY create new files or append to allowed sections only.
4. Only dev_notes.md may receive log entries.
5. Task files (3_tasks) may contain only goals, scope, status, milestones.
6. Technical designs MUST go into 4_design/.
7. API contracts and schema MUST go into 5_specs/.
8. Performance, testing, risks MUST go into 6_reports/.
9. All entries MUST include timestamps when generated by AI.
