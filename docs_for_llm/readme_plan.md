# 核聚变门户网站项目简介

## 🎯 项目目标
本项目旨在构建一个面向公众与科研工作者的「核聚变门户网站」，整合核聚变科学、工程、产业化及政策等多领域信息，为学习者、工程师与研究者提供系统化、可交互的知识平台。

最终目标是实现一个融合 **内容展示、知识管理、用户交互与AI辅助功能** 的综合门户，让更多人了解、学习并参与核聚变事业。

---

## 🧩 当前架构
- 前端：Next.js（基于 React），采用现代化 SSR/CSR 混合渲染模式  
- 后端：Node.js + Prisma + PostgreSQL  
- UI：TailwindCSS  
- 状态管理：React Hooks / Context  
- 运行方式：`npm run dev`  

---

## 📘 已实现功能
- ✅ 首页与导航结构  
- ✅ 文章展示模块（静态内容）  
- ✅ 登录页与注册页界面（未连接后端）  
- ✅ 数据库已初始化（Prisma + PostgreSQL）  

---

## 🚀 待开发功能（功能路线图）

| 模块类别 | 功能描述 | 优先级 |
|-----------|-----------|--------|
| **用户系统** | 实现注册、登录、退出登录功能，JWT 鉴权 | 🔥 高 |
| **后台管理** | 管理员可登录后台，发布/编辑文章、管理分类与标签 | 🔥 高 |
| **文章模块** | 将文章内容从数据库读取并渲染；支持 Markdown 编辑 | 🧩 中 |
| **评论与互动** | 用户可评论、点赞、收藏；支持分页加载 | 🧩 中 |
| **AI 辅助** | 为文章生成自动摘要；建立「聚变问答助手」接口 | 🌌 低 |
| **内容推荐** | 基于标签与浏览记录的个性化推荐 | 🌌 低 |
| **数据分析** | 简单的访问统计、热度榜 | 🌌 低 |

---

## 🧠 项目长期目标
1. 形成一个稳定的内容生态，支持多用户协作与投稿；
2. 引入 AI 自动生成模块（知识问答、摘要生成、聚变研究新闻摘要）；
3. 为未来的“智能核聚变研究门户”打下基础，结合科学计算、仿真展示等功能。
